# Getting started

### Requirments
  - [Pycharm](https://www.jetbrains.com/ko-kr/pycharm/download/)

### Installation
  - Clone the repository
  ```
  git clone https://github.com/choie08/OpenSource-project.git
  ```
  
  - Install dependencies
  ```
  pip install -r requirements.txt
  ```
